## 📦 Installation

### 🔧 **Requirements**
- Python 3.8+
- Git
- Internet Connection
- Linux / Termux / Kali Linux

### 💻 **Quick Install**

```bash
# Clone the repository
git clone https://github.com/Athexhacker/SN1PER.git

# Navigate to directory
cd SN1PER

# Make executable
chmod +x sn1per.sh

# Run SN1PER
./sn1per.sh
